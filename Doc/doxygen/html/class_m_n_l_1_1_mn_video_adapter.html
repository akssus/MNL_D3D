상세한 설명

작성자
Akssus

개요

하나의 MnVideoAdapter는 하나의 비디오카드 정보를 담고 있다.
비디오 카드의 이름, 비디오 카드 메모리, GPU의 최대 지원 Feature Level 의 정보를 제공한다.
