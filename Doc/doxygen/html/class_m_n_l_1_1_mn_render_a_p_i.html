개요

MnRenderAPI는 MnHardware - MnRenderAPI - MnRenderWindow 로 대표되는 MNL의 3개의 핵심 클래스 중 하나이다.
MnRenderAPI는 D3DDevice, D3DDeviceContext 인터페이스를 통해 비디오 장치와의 상호작용에 관한 모든 인터페이스를 제공한다.
MnRenderAPI는 그래픽 파이프라인의 대부분의 구간에 접근 가능하며 그래픽 파이프라인의 모든 설정은 MnRenderAPI를 통해 설정된다.
MnRenderAPI는 DrawIndexed 드로우 콜을 통해 최종 출력을 실시할 수 있다.

그래픽 파이프라인의 드로우 콜.

매개변수
indexCount설정된 인덱스 버퍼 내의 인덱스 개수.
indexOffset인덱스 버퍼 내의 시작 메모리로부터의 오프셋.
vertexBase버텍스 버퍼 내의 시작 메모리로부터의 오프셋. 인덱스를 참조해 버텍스 버퍼의 메모리로 접근하기 때문에 인덱스의 개수만 알면 된다. 그러므로 버텍스의 개수는 인자로 받지 않는다.
vertexBase는 하나의 버텍스 버퍼 안에 여러 메쉬가 선형으로 붙어 있는 경우 사용된다.
예를 들어 vertexBase가 10일 경우 인덱스 0은 실제로 인덱스 10이다.
이는 MnMesh의 SubMesh를 구현할 때 사용된다.
